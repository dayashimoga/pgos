# 📦 Release & Delivery Engine

Automatically compiles release documentation:
- **Release Notes** — Summarized changes, achievements, and contributor notes.
- **Changelog** — Git log mappings and issue references.
- **Migration Docs** — Instructions for upgrading database migrations.
- **Compatibility Matrix** — Module interfaces and adapter compatibility mappings.
- **Upgrade/Deprecation Guides** — Deprecated APIs and upgrade guidelines.
