# 🔑 Administrator's Guide

## 1. Admin Tasks
- Managing project instances and system resources.
- Adjusting global RBAC credentials and model API allocations.
- Configuring security policies and model-routing limits.

## 2. CLI Command Reference
- `guardian admin users` — Manage developer logins.
- `guardian admin roles` — Establish permission states.
- `guardian admin policy` — Deploy strict verification rules.
- `guardian admin audit` — Query validation and compliance reports.
