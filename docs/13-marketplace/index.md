# 🛒 PGOS Agent & Plugin Marketplace

Allows plug-and-play extensions for custom runtime setups:
- **Policies**: Regulatory compliance rulesets (HIPAA, SOC2).
- **Agents**: Custom Coder, Architect, and Reviewer sub-agents.
- **Validators**: Specialized security, performance, or UI validators.
- **Templates**: Code templates, monorepo starters, and Docker compose files.
- **Models**: Pre-configured configurations for local LLMs or custom routers.
- **Recovery Profiles**: Custom snapshot and fallback configurations.
- **Architecture Rules**: Modular design limits and validation rules.
