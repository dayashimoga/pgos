# 🧠 Context Engine

Converts raw project data into reusable context intelligence.
```
Repo → Parser → AST → Dependency Graph → Semantic Graph → Context Compiler → Context Package (project_context.yaml)
```
- **Inputs**: Code files, docs, tests, Git commits, prompt history, tickets, and active schemas.
- **Outputs**: `project_context.yaml` model package, vector memories, and dependency graphs.
